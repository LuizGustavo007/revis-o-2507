O que é HTML?

(HyperText Markup Language) é a linguagem padrão para a criação de páginas web. Ele descreve a estrutura de uma página web e consiste em uma série de elementos que informam ao navegador como exibir o conteúdo.

Tags HTML Básicas

Parágrafos

A tag <p> é usada para definir um parágrafo de texto. Ela agrupa blocos de texto, tornando-o fácil de ler e estruturar dentro do documento HTML.

Listas

Existem dois tipos principais de listas em HTML:

  • Listas não ordenadas cria uma lista não ordenada. Os itens da lista são geralmente apresentados com marcadores, utilizando a tag <ul>.
  • Listas ordenadas cria uma lista ordenada. Os itens da lista são apresentados em uma sequência numerada, utilizando a tag <ol>.
  • A tag <li> define um item em uma lista. Pode ser usada dentro de uma lista não ordenada (<ul>) ou ordenada (<ol>).

Exemplo de Lista não ordenada:

  • Item 1
  • Item 2
  • Item 3

Exemplo de Lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Tabelas

Tabelas são criadas utilizando as tags <table>, <tr>, <td>, e <th>.

  • <table> é usada para criar uma tabela em um documento HTML. Ela contém outras tags que definem a estrutura da tabela.
  • <tr> define uma linha em uma tabela. Ela é usada dentro da tag <table>.
  • <td> define uma célula de dados em uma linha de tabela. Ela é usada dentro da tag <tr>.
  • <th> define uma célula de cabeçalho em uma linha de tabela. As células de cabeçalho são geralmente exibidas em negrito e centradas.

Imagens

A tag <img> é usada para incorporar uma imagem em uma página HTML. A tag é auto-contida, ou seja, não tem uma tag de fechamento, e utiliza atributos para especificar a fonte (src), o texto alternativo (alt), e outras propriedades como largura (width) e altura (height).

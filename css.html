O que CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a apresentação e o layout de documentos HTML. Com CSS, você pode controlar a aparência de elementos HTML, como cores, fontes, espaçamentos e disposição na página.

Conceitos Básicos

Seletores

Os seletores em CSS são usados para aplicar estilos a elementos HTML específicos. Aqui estão alguns dos seletores mais básicos e suas aplicações:

  1. Seletores de Tag

    2. Seletores de tag são usados para aplicar estilos a todos os elementos de um determinado tipo. Por exemplo, o seletor de tag <p> aplica estilos a todos os elementos <p> (parágrafos) no documento HTML.

  2. Seletores de Classe

    3. Seletores de classe são usados para aplicar estilos a todos os elementos que têm um atributo class específico. As classes são precedidas por um ponto (.) no CSS.

  3. Seletores de ID

    4. Seletores de ID são usados para aplicar estilos a um único elemento com um atributo id específico. IDs são precedidos por uma Hashtag (#) no CSS e devem ser únicos dentro de um documento HTML.

Propriedades e Valores

Propriedades:

São os aspectos do estilo que você deseja aplicar a um elemento HTML. Cada propriedade controla um aspecto específico do estilo, como cor, tamanho da fonte, espaçamento, etc.

  • color: Define a cor do texto.
  • font-size: Define o tamanho da fonte.
  • margin: Define o espaço ao redor de um elemento.
  • padding: Define o espaço dentro de um elemento, entre o conteúdo e a borda.
  • background-color: Define a cor de fundo de um elemento.
  • border: Define a borda ao redor de um elemento.

Valores

São as quantidades ou especificações atribuídas a uma propriedade. Eles definem o aspecto visual ou o comportamento da propriedade.

  • Para a propriedade color, valores podem ser cores nomeadas (como red), valores hexadecimais (como #ff0000), ou valores RGB (como rgb(255,0,0)).
  • Para a propriedade font-size, valores podem ser unidades em pixels (16px), em pontos (12pt), ou unidades relativas (1em).
  • Para a propriedade margin, valores podem ser em pixels (20px), porcentagens (5%), ou palavras-chave (auto).
  • Para a propriedade background-color, valores podem ser cores nomeadas (blue), valores hexadecimais (#0000ff), ou valores RGBA (rgba(0,0,255,0.5)).

Classes e IDs:

Classes permitem aplicar o mesmo estilo a vários elementos HTML. Elas são úteis quando você deseja aplicar um estilo a um grupo de elementos ou quando quer reutilizar o mesmo estilo em vários locais.

  • No HTML: A classe é atribuída a um elemento usando o atributo class.
  • No CSS: A classe é selecionada usando um ponto (.) antes do nome da classe.

IDs são usados para aplicar estilos a um único elemento HTML. Eles devem ser únicos dentro de uma página, significando que cada ID deve ser usado apenas uma vez.:

  • No HTML: O ID é atribuído a um elemento usando o atributo id.
  • No CSS: O ID é selecionado usando uma cerquilha (#) antes do nome do ID.
